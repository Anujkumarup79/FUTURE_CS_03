# FUTURE_CS_03 - API Security Risk Analysis

## Description
This project analyzes API security risks such as missing authentication and data exposure.

## Tools Used
- Postman
- Browser DevTools

## Findings
- No Authentication (High)
- No Authorization (High)
- Data Exposure (Medium)

## Conclusion
The API lacks proper security controls and exposes sensitive data.

## Recommendations
- Implement authentication
- Use access control
- Apply rate limiting
- Validate inputs

This project is for educational purposes only.
